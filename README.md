# Tic-Tac-Toe Game 🎮
A classic browser-based Tic-Tac-Toe game built using HTML, CSS, and JavaScript.  
This project demonstrates DOM manipulation, game logic implementation, and interactive UI design.

## 🚀 Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation & Game Logic)

## ✨ Features
- Two-player game mode
- Dynamic turn switching (X and O)
- Winner detection logic
- Draw detection
- Reset / New Game functionality
- Responsive desige
  
## 🧠 What I Learned
- Implementing game logic using JavaScript
- Handling click events
- Updating UI dynamically using DOM
- Managing game state with variables and conditions
